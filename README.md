# 🔢 8. Сұрыптау визуализациясы (Sorting Visualization)

## 🎯 Мақсаты:
Бұл жобаның мақсаты — сұрыптау алгоритмдерін түсіну және олардың жұмыс істеу принципін визуалды түрде көру.

---

## 📌 Жобаның сипаттамасы:
Бағдарлама кемінде екі сұрыптау алгоритмін жүзеге асырады:
- Bubble Sort  
- Selection Sort  

Әр қадам консоль немесе GUI арқылы көрсетіледі. Бұл массивтің қалай өзгеретінін бақылауға мүмкіндік береді.

---

## ❓ Сұрақтар және жауаптар:

### 1. Bubble sort қалай жұмыс істейді?
Bubble sort көршілес элементтерді салыстырып, орындарын ауыстыру арқылы жұмыс істейді. Әр айналым сайын ең үлкен элемент массивтің соңына жылжиды ("жүзіп шығады").

---

### 2. Алгоритм күрделілігі деген не?
Алгоритм күрделілігі — алгоритмнің уақыт пен ресурс (жад) жағынан тиімділігін көрсетеді. Ол көбінесе O(n), O(n²) сияқты Big-O нотациясымен өрнектеледі.

---

### 3. Қай сұрыптау тиімді?
Қарапайым алгоритмдер ішінде:
- Bubble Sort — баяу (O(n²))  
- Selection Sort — сәл тиімдірек, бірақ ол да O(n²)  

Үлкен деректер үшін Quick Sort немесе Merge Sort сияқты алгоритмдер әлдеқайда тиімді.

---

## 📊 Қорытынды:
Бұл жоба сұрыптау алгоритмдерінің жұмысын түсінуге көмектеседі. Әр қадамды визуализациялау арқылы алгоритм логикасын терең меңгеруге болады.
